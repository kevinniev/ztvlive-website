## Installation

To get started with the ztvlive-website project, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/kevinniev/ztvlive-website.git
   ```
2. Navigate to the project directory:
   ```bash
   cd ztvlive-website
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage

To start the development server, run:
```bash
npm start
```

Visit `http://localhost:3000` in your browser to see the website in action.
