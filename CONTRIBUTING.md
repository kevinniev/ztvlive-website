# Contributing to ZTVLive Website

Thank you for considering contributing to the ZTVLive Website! We appreciate your effort to improve our project. Please follow the guidelines below to help streamline the contribution process.

## How to Contribute
1. **Fork the Repository**: Click the "Fork" button on the top right corner of the repository page.
2. **Clone Your Fork**: Clone your forked repository to your local machine using:
   ```
   git clone https://github.com/<your-username>/ztvlive-website.git
   ```
3. **Create a Branch**: Create a branch for your feature or bug fix:
   ```
   git checkout -b feature/my-feature
   ```
4. **Make Changes**: Make your changes and commit them with a descriptive message:
   ```
   git commit -m "Add new feature"
   ```
5. **Push Changes**: Push your changes to your forked repository:
   ```
   git push origin feature/my-feature
   ```
6. **Create a Pull Request**: Go to the original repository and create a pull request.

## Guidelines
- **Code Standards**: Ensure your code adheres to the existing coding style and standards in the codebase.
- **Testing**: If applicable, ensure that your contributions are covered by tests.
- **Documentation**: Update documentation as necessary when making changes that affect users or usage.

## Reporting Issues
If you encounter any issues, please report them using the [issues section](https://github.com/kevinniev/ztvlive-website/issues) of the repository.
